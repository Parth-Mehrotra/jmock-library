Building from Source

You can build jMock with Ant3. You will need to install Ant version > 1.7. Running the ant command in the root of the source tree will perform compile, test and build the jMock JARs, generate Javadoc and package it all into ZIP archives. The built artefacts are left in the build subdirectory.
